## Installation
  You need to have the following <br>
  <a href="https://nodejs.org/en/">nodejs</a> Only for beta release
  open terminal
  ```
  git clone https://github.com/DANNY-05/Elevate-Music-App.git
  ```
```
  cd Elevate-Music-App
```
```
  npm install 
```
  ``` 
  npm start
  ```
